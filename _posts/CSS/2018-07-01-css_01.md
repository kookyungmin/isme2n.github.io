## CSS 개요

<br>
<br>
XHTML, HTML5는 웹 문서의 구조를 만드는 것이라면,
<br>
<br>
CSS는 디자인입니다.
<br>
<br>
글꼴이나 색상을 변경하거나, 배경을 넣거나, 문단들을 원하는대로 배치하는 작업 등을 CSS를 이용하여 합니다.
<br>
<br>
그림을 그려서 표현하는 디자인이 아니고 명령을 입력하여 디자인하는 것인데,
<br>
<br>
앞으로 저희가 알아볼 내용이 바로 그것입니다.
