## LED 켜기

<br>
<br>
아두이노에 LED 전구를 연결하여 껐다켜보겠습니다~!
<br>
<br>
<br>

```
int ledPin = 13;
void setup() {
  pinMode(ledPin,OUTPUT);
}

void loop() {
  flash();  

}

void flash(){
  digitalWrite(ledPin,HIGH);
  delay(1000);
  digitalWrite(ledPin,LOW);
  delay(1000);
}
```
